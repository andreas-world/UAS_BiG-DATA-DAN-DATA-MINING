## Dataset

Dataset yang digunakan merupakan **dataset sekunder** yang bersumber dari **Kaggle**, yaitu *TMDB Movie Metadata*. Dataset ini terdiri dari dua file utama, yaitu:

- `tmdb_5000_movies.csv`
- `tmdb_5000_credits.csv`

Dataset tersebut berisi informasi film seperti **budget, revenue, popularity, dan vote_average** yang digunakan sebagai dasar dalam proses analisis dan clustering film.

---

## Objectives

Project ini bertujuan untuk:
- Memahami karakteristik data film menggunakan Exploratory Data Analysis (EDA)
- Melakukan preprocessing data untuk meningkatkan kualitas data
- Melakukan seleksi fitur yang relevan untuk proses clustering
- Mengimplementasikan algoritma **K-Means Clustering**
- Mengevaluasi hasil clustering menggunakan metode evaluasi yang sesuai
- Menginterpretasikan hasil cluster berdasarkan karakteristik film
